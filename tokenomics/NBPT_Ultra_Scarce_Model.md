## Security & Compliance

### ERC-3643 Features
- Identity-gated transfer restrictions
- zkSBT accredited investor verification
- Modular compliance engine (Reg D, OFAC, Rule 144)
- Agent-controlled forced transfers for regulatory compliance

### Regulatory Framework (Pending Counsel Review)
- SEC Rule 506(b) / 506(c) exemption analysis required
- Form D filing required before any offering
- Blue Sky notice filings per state
- Transfer restrictions enforced on-chain
- KYC/AML via approved vendor
- Accredited investor verification required

### Hard-Blocked Operations
- Autonomous treasury movements: **BLOCKED**
- Autonomous securities operations: **BLOCKED**
- New York virtual currency activity: **BLOCKED** pending BitLicense review
- Public secondary market: **BLOCKED** pending ATS/exchange analysis

Invoice → Closeout) launches independently of any token activity.
